Making weekly content? Use this project, new chat session.

Weekly pieces are built right here in the weeks/ folder — they depend on this project’s CSS, fonts, and logos. A separate project breaks the HTML preview. Each weekly session is a fresh conversation with short history; old turns are trimmed so context stays lean.

  1. Start the chat here with “start q#-w##” (e.g. start q2-w14). That’s the only trigger needed — no setup, no template descriptions.
  2. Tell it which template number (1, 2, or 3) per piece, paste your copy, and drop images one at a time.
  3. Review the HTML preview in weeks/q#-w##.html. Request any edits before approving.
  4. Say “export it” — you’ll get a zipped folder of full-resolution JPGs to download.
